# 📚 欢迎使用我的 Wiki

这是一个基于 **Docsify** 构建的知识库系统。

## ✨ 特性

- ✅ **纯静态** - 无需构建，直接部署
- ✅ **实时渲染** - Markdown 即时预览
- ✅ **侧边栏折叠** - 支持展开/收起导航
- ✅ **全局搜索** - 快速查找文档
- ✅ **响应式设计** - 适配手机/平板/电脑
- ✅ **代码高亮** - 支持多种编程语言
- ✅ **图片缩放** - 点击放大查看
- ✅ **一键复制** - 代码块支持复制

## 🚀 快速开始

1. **浏览文档** - 点击左侧导航栏查看内容
2. **搜索功能** - 使用顶部搜索框快速查找
3. **折叠导航** - 点击 ▼ 图标展开/收起目录

## 📖 文档结构

- **快速开始** - 安装和配置指南
- **使用指南** - 详细使用说明
- **API 文档** - 接口和插件开发
- **进阶教程** - 高级功能和优化

## 🔧 技术栈

- [Docsify](https://docsify.js.org/) - 静态文档生成器
- [GitHub Pages](https://pages.github.com/) - 免费托管
- [Markdown](https://www.markdownguide.org/) - 轻量级标记语言
