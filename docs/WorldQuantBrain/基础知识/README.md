# 📚 WorldQuant BRAIN 基础知识指南

欢迎来到 WorldQuant BRAIN 量化交易平台的**基础知识**模块！本模块为初学者和进阶用户提供全面的入门指南，帮助您快速掌握量化交易的核心概念和平台使用技巧。

### 🎯 模块目标

通过本模块的学习，您将能够：
- 理解 WorldQuant BRAIN 平台的运行机制
- 掌握量化交易的基本概念和术语
- 熟悉平台的核心指标和算子
- 构建您的第一个 Alpha 策略

### 💡 学习建议

- **理论与实践结合**：边学边在平台上动手操作
- **从小策略开始**：从简单的单因子策略入手，逐步复杂化
- **关注风险控制**：始终将风险管理放在首位
- **多做回测**：通过大量回测验证策略的稳健性
