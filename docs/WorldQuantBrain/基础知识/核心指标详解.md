## 🔑 核心指标速览表

| 指标 | 公式/定义 | 目标值 | 关键解读 |
|------|-----------|--------|----------|
| **Sharpe** | `年化收益率 / 年化收益率标准差`<br>（日收益×√250年化） | `> 1.0` ✅<br>`> 1.5` 🎯 | 衡量**风险调整后收益**，值越高说明收益越稳定、波动越小 |
| **Turnover** | `当日交易额 / 持仓总额` | `< 0.7` ⚠️（平台上限）<br>`< 0.4` 🎯（理想） | 换手率，过高会增加交易成本侵蚀净收益，过低可能信号迟钝 |

| **Drawdown** | 回测期内任意年度最大亏损百分比 | `<br 15%` ⚠️ | 风险控制指标，健康标准：`Returns / Drawdown > 1` |

## 核心概念定义
### 1.Alpha（阿尔法）
✅ 定义：预测金融资产未来价格变动的数学模型</br>
✅ 本质：通过量化规则提前定义"何时、买多少、买哪些"</br>
✅ 目标：在控制风险的前提下，获取超越市场的稳定收益
### 2. Backtesting（回测）
✅ 定义：用历史数据验证量化策略有效性的过程</br>
✅ 关键原则：</br>
   • 避免未来函数（只能用T-1日数据交易T日）</br>
   • 考虑交易成本与滑点</br>
   • 覆盖完整牛熊周期（建议≥252交易日）
### 3. Long-Short Portfolio（多空组合）
✅ 机制：同时做多高分股票 + 做空低分股票</br>
✅ 优势：</br>
• 对冲市场系统性风险（Market Neutral）</br>
• 收益来源纯粹来自选股能力（Alpha）</br>
✅ 注意：总投入资金 = 账面资金的一半（多空各占50%）

## 六大核心指标详解
### 1️⃣ Sharpe Ratio（夏普比率）
| 属性 | 内容 |
|------|-----------|
| 定义 | 风险调整后的收益衡量指标 |
| 公式 | `Sharpe = 平均年化收益率 / 年化收益率标准差` |
| 计算细节 | 日收益需×√250年化 |
| 目标值 | > 1.0（合格），> 1.5（优秀） |
| 优化方向 | ↑ 收益稳定性，↓ 收益波动率 |
### 2️⃣ Turnover（换手率）
| 属性 | 内容 |
|------|-----------|
| 定义 | 每日交易资本占总持仓的比例 |
| 公式 | `Turnover = 当日交易额 / 持仓总额` |
| 目标值 | <br 0.7（平台上限），<br 0.4（理想） |
| 影响 | 换手越高 → 交易成本越高 → 净收益越低 |
| 优化方向 | 信号衰减、降低更新频率、设置变化阈值 |
### 3️⃣ Fitness（适应度）⭐ 核心优化目标
| 属性 | 内容 |
|------|-----------|
| 定义 | 综合评估Alpha质量的加权指标 |
| 公式 | `Fitness = Sharpe × √\|Returns\| / Max(Turnover, 0.125)` |
| 设计逻辑 | 鼓励：高夏普+高收益+低换手；惩罚：高换手 |
| 目标值 | > 0.3（教程），> 0.5（提交），> 1.0（优秀） |
| 优化方向 | "提夏普、保收益、压换手"|
### 4️⃣ Returns（收益率）
| 属性 | 内容 |
|------|-----------|
| 定义 | 回测期内多空组合的总收益率 |
| 注意 | 	• 是未年化的原始收益</br>• 基于多空组合（非单边做多）</br>• 正值为盈利，负值为亏损 |
| 目标值 | > 0%（基础），> 5%（优秀） |
| 警惕 | 高收益+高回撤 = 高风险策略，需结合Drawdown评估 |
### 5️⃣ Drawdown（最大回撤）
| 属性 | 内容 |
|------|-----------|
| 定义 | 回测期内任意年度出现的最大亏损百分比 |
| 意义 | 衡量策略的极端风险承受能力 |
| 健康标准 | `Returns / Drawdown > 1`（收益能覆盖最大亏损） |
| 优化方向 | 加入防御因子（低波动、高质量）、行业中性化 |
### 6️⃣ Margin（边际收益）
| 属性 | 内容 |
|------|-----------|
| 定义 | 每单位交易额产生的盈亏 |
| 公式 | `Margin = 总PnL / 总交易额` |
| 单位 | 基点（basis points, ‱），1‱ = 0.01% = 万分之一 |
| 示例 | Margin = 50‱ → 每交易1万元，盈利5元 |
| 目标值 | > 5‱（基础），> 20‱（优秀） |
