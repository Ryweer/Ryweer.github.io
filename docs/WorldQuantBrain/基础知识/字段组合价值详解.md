# 字段组合价值详解

## 文档结构概览
本文件包含以下三个核心知识主题：
1. `equity/assets`：股东权益与总资产比率
2. `operating_income/cap`：营业利润与市值的基本面/市场对比
3. `EV/EBITDA`：企业价值与 EBITDA 的估值倍数

---

## 1. equity/assets

### 定义
`equity/assets` 表示股东权益占总资产的比率。该比率用于衡量公司的财务结构、杠杆水平和财务稳健性。

### 计算方法
- **分子 (Equity, 股东权益)** ：公司总资产中归属于股东的部分，即净资产（总资产 - 总负债）。
- **分母 (Assets, 总资产)** ：公司拥有或控制的所有经济资源的总和。
- **比率 (Equity/Assets)** ：表示股东权益在总资产中所占比例。

### 业务意义
- 比率越高，表明公司资产中由股东出资部分越多。
- 这通常意味着财务杠杆较低，资产负债结构更稳健。

### 在 Alpha 构建中的应用
- 该指标可作为简单的基本面选股因子。
- 逻辑：选择股东权益占比较高、财务风险较低的公司。
- 在某些市场环境中，这类公司可能表现更稳定。

### 相关名称
- 权益比率（Equity Ratio）
- 自有资本比率

其倒数 `Assets/Equity` 与财务杠杆直接相关。

### 小结
`equity/assets` 是衡量公司资本结构和财务风险的基础财务比率，可作为量化因子的基本构件。

## 2. operating_income/cap

### 数据类型与特征对比
- **基本面数据**（如营业利润）：更新频率低（季度/年度），反映公司实际经营成果和内在价值。
- **市场价值数据**（如市值）：更新频率高（每日），反映市场主观估值和预期。

### Alpha 构建核心思想
- 核心目标是将公司内在价值指标与市场预期指标进行对比。
- 通过比较营业利润与市值，可以判断公司是否“业绩支撑估值”或“估值高于预期”。

### 数据比较方法论
- **减法**：适用于同尺度、同单位数据，关注绝对差值。
- **除法**：适用于不同尺度或单位数据，关注相对比例或比率。
- **其他形式**：如 `(y - x) / x`，表示变化率或相对差异。

### 关键 Alpha 表达式与含义
- `operating_income / cap`：课程标准答案。
- 含义：对比公司营业利润与市值，衡量单位市值产生的盈利能力。
- 比值越高，可能意味着公司盈利效率较高、相对估值较低。

### 实践注意事项
- **数据尺度与单位**：组合不同来源数据时要标准化，避免公司规模差异导致结果失真。
- **信号强度**：初步 Alpha 信号可能较弱，需要进行数据清洗、标准化和参数优化。

### 小结
`operating_income/cap` 体现了基本面现实与市场定价预期之间的对比，为选股或排序提供潜在因子。

## 3. EV/EBITDA：价格-基本面结合型 Alpha

### 背景说明
本节讨论如何用价格成交量（PV）数据与基本面数据结合，形成更具意义的 Alpha。

### 关键指标定义
- **EBITDA**：息税折旧摊销前利润，反映公司核心经营盈利能力。
  - 更新频率低：季度/年度报告。
  - 提供更清晰的运营盈利视图。
  - 有利于比较不同资本结构或税率的公司。

- **企业价值（EV）**：公司的总收购价值。
  - 计算公式：EV = 市值 + 总债务 + 少数股东权益 + 优先股 - 现金及现金等价物。
  - 更新频率高：随股价变动。
  - 相较于市值更全面，考虑了债务与现金状况。

### Alpha 思路
- 企业价值代表市场对公司总价值的评估。
- EBITDA 代表公司持续经营的盈利能力。
- 通过比较这两者，可以判断公司是否相对于盈利能力被低估或高估。

### 关键比率：EV/EBITDA
- `EV / EBITDA`：常称为企业价值倍数。
- 含义：按公司当前盈利水平，需要多少年才能“赚回”公司的总价值。
- 越低通常意味着公司可能被低估；越高可能意味着估值较高或市场对增长的预期较高。

### 为什么除法有意义
- 允许不同规模公司之间的比较。
- 形成标准化指标，便于跨公司评估。
- 提供衡量公司盈利能力与估值之间关系的量化方式。

### 应用与注意事项
- **行业比较重要**：不同产业 EV/EBITDA 典型水平差异大，需在行业内进行排名比较。
- **比率局限性**：EBITDA 未考虑资本支出和营运资本变动，对资本密集型行业适用性有限。

### 小结
`EV/EBITDA` 是典型的价格-基本面结合型估值因子，体现了从会计数据到可投资信号的转化过程。

---

## 4. 核心知识点总结

### 关键财务指标与特征
- **equity/assets**：衡量资本结构和财务稳健性。
- **operating_income/cap**：衡量盈利能力与市场价值之间的关系。
- **EV/EBITDA**：衡量企业价值与核心盈利能力之间的估值水平。

### Alpha 构建逻辑
- 基本面指标与市场估值指标的对比，是常见的 Alpha 构建方式。
- 除法比率是连接不同单位数据、实现跨公司可比性的主要手段。

### 实践注意点
- 标准化数据、处理单位差异、避免规模偏差。
- 行业比较比全市场比较更合理。
- 了解不同指标适用范围，避免盲目套用。
